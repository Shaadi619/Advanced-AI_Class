# Multimodal AI Group Class Project

This repository contains a short Advanced AI group class project exploring multimodal generation with two pretrained models:

- GPT-2 for text generation
- Stable Diffusion for image generation

The project notebook demonstrates how a single idea or prompt can be used across different AI modalities. GPT-2 generates text from the prompt, while the Stable Diffusion model creates an image based on the same concept.

## Project File

- `notebooks/Advanced_Ai_Group_Presentation_Shaadi_Awab_Abdallah.ipynb`

## Goal

The goal of this project is to show how language and image generation models can work together in a simple multimodal AI workflow for a class presentation.
