# Demo Script

Use this flow for the required `3-5` minute demo:

1. Introduce the project as a multimodal assistant that combines an instruction-following LLM, Stable Diffusion, and human feedback.
2. Enter a shared prompt such as `A futuristic smart city with flying cars`.
3. Click `Generate Text and Image`.
4. Show the three generated text responses and explain that they come from the LLM.
5. Show the generated image and explain that it uses the same prompt in Stable Diffusion.
6. Rate each response from `1-5` and choose the best one.
7. Submit the feedback and show the ranking table updating.
8. Explain that the ranking table is the simplified RLHF reward signal for the system.
9. Close by mentioning the main limitation: feedback is stored and ranked, but the model itself is not retrained.
